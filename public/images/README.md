# Images Directory

This directory is for storing images used in your portfolio.

## Required Images

### Profile Image
- **File name**: `profile.jpg` or `profile.png`
- **Location**: `public/images/profile.jpg` or `public/images/profile.png`
- **Usage**: Displayed on the About page
- **Recommended size**: 512x512 pixels (square image)
- **Format**: JPG or PNG

## Optional Images

### Project Images
If you want to add screenshots or images for your projects, you can add them here:
- `project-1.jpg` - For Construction Domain Project
- `project-2.jpg` - For Retail Domain Project (Backend)
- `project-3.jpg` - For Retail Domain Project (Full Stack)
- `project-4.jpg` - For Law Domain Project

## How to Add Images

1. **Profile Image**:
   - Take or find a professional headshot photo
   - Resize it to 512x512 pixels (square)
   - Save it as `profile.jpg` or `profile.png`
   - Place it in this directory: `public/images/profile.jpg`

2. **Project Images** (optional):
   - Take screenshots of your projects
   - Resize them to a reasonable size (e.g., 1200x800 pixels)
   - Save them with descriptive names
   - Place them in this directory

## Image Optimization Tips

- Use JPG for photos (smaller file size)
- Use PNG for graphics with transparency
- Compress images before adding them (use tools like TinyPNG or ImageOptim)
- Keep file sizes under 500KB for better performance

## Note

If you don't add a profile image, the About page will automatically show your initials (HSC) in a styled circle.
